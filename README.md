# Project & Task Management System

An advanced client project, frontend prototype with a robust JavaScript engine, rich modern design, and integrated DevOps workflow.

## Features
- **Login UI**: Validated user authentication.
- **Project Dashboard**: Key performance metrics (Total Projects, Tasks Status, Overdue Tasks).
- **Task Creation Module**: Form with date-validation and priority highlighting.
- **Advanced Logic**: Search by employee, status/priority filter, Local Storage persistence.
- **Reports Section**: Project progress bars, overdue counts, print summaries, and data export.
- **Theme Customization**: Beautiful Light & Dark mode support.
